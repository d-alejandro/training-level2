Что выведет программа? Объяснить вывод программы.

```go
package main

import (
	"fmt"
	"math/rand"
	"time"
)

func asChan(vs ...int) <-chan int {
	c := make(chan int)

	go func() {
		for _, v := range vs {
			c <- v
			time.Sleep(time.Duration(rand.Intn(1000)) * time.Millisecond)
		}

		close(c)
	}()
	return c
}

func merge(a, b <-chan int) <-chan int {
	c := make(chan int)
	go func() {
		for {
			select {
			case v := <-a:
				c <- v
			case v := <-b:
				c <- v
			}
		}
	}()
	return c
}

func main() {

	a := asChan(1, 3, 5, 7)
	b := asChan(2, 4 ,6, 8)
	c := merge(a, b )
	for v := range c {
		fmt.Println(v)
	}
}
```

Ответ:
```
1
2
4
6
3
5
8
7
0
0
0 ... бесконечное повторение вывода 0 
```
```
Программа выводит числа при чтении канала c, который был получен как значение функции merge().
Сначала дважды вызывается функция asChan(), в которую на вход передаются числа, а на выходе отдаётся канал. Далее эти
два канала передаются на вход функции merge(). Функция asChan() получает данные из аргументов функции, создаётся канал
и внутри горутины пишутся входящие данные в канал, после чего канал закрывается. В функции merge() создаётся канал
для отправки в main(), создаётся горутина, внутри которой в бесконечном цикле размещена select конструкция.
В select конструкции происходит чтение из входящих каналов от аргументов функции и дальнейшая отправка данных в
выходной канал. После чтения и отправки всех чисел из asChan() каналы закроются и в конструкцию select будут приходить
нулевые значения каналов, то есть нули. Данный процесс будет происходить в бесконечном цикле, что соответственно будут
бесконечно отправляться нули в выходной канал c.
```
