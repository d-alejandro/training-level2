Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
2
1
```
```
В первой строке вывода распечатан результат значения функции test(). Эта функция содержит именованный параметр
результата вывода в виде переменной x целочисленного типа. Начальное значение переменной равно 0, далее переменной
присваивается значение 1. В конце работы функции отрабатывает вызов оператора defer, который выполняет анонимную
функцию. Внутри анонимной функции происходит инкремент x на 1. Так как defer имеет доступ и может изменять
именованные параметры результата, то при выходе из функции test() значение x будет 2.

Во второй строке вывода распечатан результат значения функции anotherTest(). Переменная x объявлена внутри функции,
начальное значение равно 0. Это значение захватывается в область видимости анонимной функции при декларации
оператора defer. Далее переменной x присваивается единица. После срабатывает вызов оператора defer, внутри анонимной
функции происходит инкремент переменной x на 1, в результате будет 1. Далее полученный результат в анонимной функции
отбрасывается и происходит возврат из anotherTest() с ранее полученным значением x перед вызовом defer равным 1.

Оператор defer вызывается при достижении конца функции, либо при достижении оператора return, либо при вызове panic
горутины. При наличии нескольких defer вызов осуществляется в обратном порядке их деклараций, то есть в порядке
работы стэка: последняя добавленная функция будет вызвана первой.
```
