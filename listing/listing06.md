Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
[3 2 3]
```
```
Программа распечатает содержимое слайса из переменной s. Перед выводом слайс передаётся на вход функции modifySlice().
Внутри функции первый элемент слайса переменной i заменяется на 3, а так как не было релокации, то переменная i
и переменная s ссылаются на один и тот же участок памяти содержащий массив, следовательно изменение будет доступно для
двух переменных. Далее происходит append слайса для переменной i, после чего произойдёт релокация слайса на новый
участок памяти, что приведёт к копированию всей структуры слайса с увеличение ёмкости и добавлением 4 в конец слайса.
Последующие манипуляции со слайсом в переменной i не затронут слайс, на который ссылается переменная s из главной
функции.

Слайсы внутри представляют собой структуру slice, которая содержит свойство array с типом unsafe.Pointer,
свойство len с типом int и свойство cap с типом int.
При передаче слайса в качестве аргумента функции длина len и вместимость cap передаются по значению, а массив
значений передается по ссылке. 
```
